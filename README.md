# 0 DISTRICT — 0PromoRecords

The digital home of African music. A platform for Artists, Labels, and Curators operating at a global standard.

**Live at:** [0promorecords.com](https://0promorecords.com)

---

## What it is

0 District is a full-stack music platform built for the African music ecosystem — Afrobeats, Amapiano, Highlife, Afro Soul, Afro House, and every sound in between.

Three types of Operators run the District:

- **Artists** — Upload tracks, track plays and revenue, find collaborators, pitch to editorial
- **Labels** — Manage full rosters, schedule releases, run the deal pipeline
- **Curators** — Build playlists, manage editorial queues, publish essays, run featured picks

---

## Stack

- **Frontend** — Vanilla HTML/CSS/JS, IBM Plex Mono + Barlow Condensed
- **Backend** — Supabase (Auth, PostgreSQL, RLS, Edge Functions)
- **Deploy** — Netlify (CI/CD from this repo)
- **Radio** — Spotify Web Playback SDK (Premium)
